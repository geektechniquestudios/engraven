## Keeping It True

[[Memory Hygiene and Verification]] catalogs the ways memory rots (broken
links, index drift, budget overflow, stale facts), the two linters that make
rot loud, and the write-time habits that prevent most of it. Read it when a
linter goes red or memory "feels off."

## Keeping the Story

[[Session Archiving]] covers temporal memory: archiving each significant
session's decisions, failures, and open threads so future agents inherit
the *why*, not just the diff. Entries live in `Session-Archive/`, indexed by
[[Session Archive Index]].
